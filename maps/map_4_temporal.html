1 Introduction

This document presents Map 4 of the DigiKat project, analyzing the temporal dynamics of Croatian Catholic digital media. The core question driving this analysis is How does Croatian Catholic digital communication evolve over time, and how does it respond to events?

Understanding temporal patterns reveals the rhythms of religious communication online, from daily posting cycles to annual liturgical seasons. This analysis examines over 600,000 posts from 2021 to 2025 to uncover when Croatian Catholics communicate, how patterns shift across platforms, and how the digital sphere responds to significant religious and secular events.

3 Analysis 4.2 Liturgical Calendar Effects

Croatian Catholic digital communication is expected to follow liturgical rhythms. This analysis maps posting patterns to the Catholic liturgical calendar to identify seasonal effects.

Show code 
calculate_easter <- function(year) {
  a <- year %% 19
  b <- year %/% 100
  c <- year %% 100
  d <- b %/% 4
  e <- b %% 4
  f <- (b + 8) %/% 25
  g <- (b - f + 1) %/% 3
  h <- (19 * a + b - d - g + 15) %% 30
  i <- c %/% 4
  k <- c %% 4
  l <- (32 + 2 * e + 2 * i - h - k) %% 7
  m <- (a + 11 * h + 22 * l) %/% 451
  month <- (h + l - 7 * m + 114) %/% 31
  day <- ((h + l - 7 * m + 114) %% 31) + 1
  as.Date(paste(year, month, day, sep = "-"))
}

assign_liturgical_season <- function(date) {
  year <- year(date)
  easter <- calculate_easter(year)
  
  ash_wednesday <- easter - 46
  palm_sunday <- easter - 7
  pentecost <- easter + 49
  
  advent_start <- as.Date(paste(year, "11", "27", sep = "-"))
  advent_start <- advent_start + (7 - lubridate::wday(advent_start) + 1) %% 7
  if (advent_start > as.Date(paste(year, "12", "03", sep = "-"))) {
    advent_start <- advent_start - 7
  }
  
  christmas <- as.Date(paste(year, "12", "25", sep = "-"))
  epiphany <- as.Date(paste(year + 1, "01", "06", sep = "-"))
  baptism_of_lord <- epiphany + (7 - lubridate::wday(epiphany)) %% 7 + 1
  if (lubridate::wday(epiphany) == 1) baptism_of_lord <- epiphany + 7
  
  prev_year_christmas <- as.Date(paste(year - 1, "12", "25", sep = "-"))
  prev_epiphany <- as.Date(paste(year, "01", "06", sep = "-"))
  
  if (date >= advent_start) {
    if (date < christmas) return("Advent")
    else return("Christmas")
  }
  
  if (date < prev_epiphany + 8 && month(date) == 1 && day(date) <= 13) {
    return("Christmas")
  }
  
  if (date >= ash_wednesday && date < easter) {
    return("Lent")
  }
  
  if (date >= easter && date < pentecost) {
    return("Easter")
  }
  
  return("Ordinary Time")
}

8.2 Key Findings

8.2.1 Overall Temporal Patterns

The corpus reveals macro trends in Croatian Catholic digital communication across multiple years. Monthly and yearly aggregations show whether the digital sphere is expanding or contracting, while peak identification highlights periods of exceptional activity.

8.2.2 Liturgical Calendar Effects

Croatian Catholic digital communication follows liturgical rhythms. The Christmas and Easter seasons show distinct patterns, and major feast days generate measurable volume increases. This confirms that religious calendar structures digital communication patterns.

8.2.3 Event Response

Major events generate different response profiles depending on their nature. Youth events like SHKM produce concentrated spikes, Vatican pronouncements may generate sustained discussion, and political events reveal engagement from specific actor segments.

8.2.4 Platform Migration

Platform composition shifts over time reveal structural changes in where Croatian Catholic digital communication occurs. Growth and decline patterns help identify which platforms are gaining importance and which are fading.

8.2.5 Posting Rhythms

Micro temporal patterns reveal distinct rhythms between institutional and personal accounts. Institutional actors follow office hour patterns while individuals post more in evenings and weekends. Sunday morning represents a distinctive Catholic digital rhythm.

8.2.6 Information Cascades

Platform activity correlations and content origin patterns suggest how information flows through the ecosystem. Understanding these patterns helps identify which platforms serve as originators versus amplifiers of Catholic digital content.

9 Appendix A Technical Notes

9.1 Liturgical Calendar Algorithm

The liturgical season assignment uses the Computus algorithm to calculate Easter dates. Seasons are assigned as follows:

Advent begins four Sundays before Christmas and runs until December 24. Christmas begins December 25 and extends through Epiphany and the Baptism of the Lord. Lent begins on Ash Wednesday (46 days before Easter) and ends on Holy Thursday. Easter begins on Easter Sunday and extends through Pentecost (50 days). Ordinary Time fills the remaining periods.

9.2 Feast Day Effect Calculation

Feast day effects are calculated by comparing a three day window (day before, feast day, day after) against a yearly baseline excluding the feast window. The effect size represents the percentage difference from baseline daily volume.

9.3 Platform Growth Calculations

Compound Annual Growth Rate (CAGR) is calculated using the standard formula assuming geometric growth between first and last observed years in the corpus.
